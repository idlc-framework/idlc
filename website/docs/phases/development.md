# Development Phases

The development phases cover the full cycle of creating, validating, and shipping infrastructure code.

| Phase | Focus |
|:------|:------|
| [Plan]({{ "/docs/phases/plan.html" | relative_url }}) | Define requirements, architecture, and resource dependencies |
| [Code]({{ "/docs/phases/code.html" | relative_url }}) | Write modules, solutions, and deployments |
| [Build]({{ "/docs/phases/build.html" | relative_url }}) | Organize and structure IaC into the repository model |
| [Test]({{ "/docs/phases/test.html" | relative_url }}) | Validate infrastructure code before deployment |
| [Release]({{ "/docs/phases/release.html" | relative_url }}) | Version and publish modules and solutions |
| [Deploy]({{ "/docs/phases/deploy.html" | relative_url }}) | Apply solutions into target environments |
