# IDLC Phases

The IDLC framework defines 8 phases organized into two categories:

```
Development:  Plan → Code → Build → Test → Release → Deploy
Operations:   Operate → Monitor
```

The Development phases cover the full cycle of creating, validating, and shipping infrastructure code. The Operations phases close the loop with day-2 management and observability that feeds back into the next development cycle.
